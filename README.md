# magnifier

> Magnifier an image

## Install

```sh
npm install --save magnifier
```

## Usage

```js
var Magnifier = require('magnifier');
new Magnifier('.image');
```

## API

### new Magnifier(image)

Create a new `Magnifier` instance

### .height(h)

Set lens height

### .width(h)

Set lens width

### .backgroundColor(color)

Set lens background color

### .borderColor(color)

Set lens border color

### .borderRadius(r)

Set lens radius

### .borderWidth(w)

Set lens border width

### .className(name)

Set lens class name

### .show()

Forced show lens

### .hide()

Forced hide lens
